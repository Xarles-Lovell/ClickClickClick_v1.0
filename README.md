# 点点点

一个功能强大、界面简洁的Windows自动连点器程序。

## ✨ 功能特性

- 🖱️ **多点位置管理** - 记录和管理多个点击位置，支持循环点击
- ⚙️ **灵活的点击设置** - 可调节点击频率（0.1-10秒或10-1000毫秒）
- 🎯 **精确的鼠标控制** - 支持左键/右键点击，使用Windows API确保兼容性
- 💾 **预设系统** - 保存和加载不同的配置预设
- ⌨️ **全局快捷键** - 自定义快捷键快速记录位置和控制连点
- 🎨 **简洁界面** - 灰色主题，清晰易用
- 📊 **实时状态显示** - 滚动日志显示操作状态
- 🎯 **连点测试** - 内置测试窗口，方便验证功能

## 📦 下载和安装

### 方式一：直接运行（推荐）

1. 下载 `release/点点点_v1.0.exe`
2. 双击运行即可使用

### 方式二：从源码运行

1. 确保已安装 Python 3.7+
2. 安装依赖：
```bash
pip install -r requirements.txt
```
3. 运行程序：
```bash
python 点点点.py
```

## 🚀 快速开始

### 基本使用流程

1. **设置点击频率** - 使用滑块或输入框调整点击间隔
2. **记录位置** - 点击"记录鼠标位置"按钮或使用快捷键（默认Alt+R）
3. **开始连点** - 点击"开始连点"按钮
4. **停止连点** - 按Esc键或点击"停止连点"按钮

### 快捷键说明

- **Alt+R** - 记录当前鼠标位置（可自定义）
- **Esc** - 停止连点
- **Ctrl+T** - 切换窗口置顶

### 预设管理

- **保存预设** - 将当前设置保存为预设，方便下次使用
- **加载预设** - 从下拉菜单选择已保存的预设
- **重命名/删除** - 管理已保存的预设

## 📖 详细功能

### 点击设置

- **频率模式**：
  - 秒/次模式：0.1-10.0秒间隔
  - 毫秒/次模式：10-1000毫秒间隔
- **鼠标按键**：左键或右键
- **点击方法**：优先使用Windows API，确保游戏兼容性

### 位置管理

- **记录位置**：支持记录多个点击位置
- **自定义命名**：为每个位置设置易识别的名称
- **重命名位置**：随时修改位置名称
- **删除位置**：双击列表项或使用删除按钮

### 窗口功能

- **窗口置顶**：使用Ctrl+T切换，标题栏显示📌图标
- **连点测试**：内置测试窗口，显示点击动画和教程
- **状态日志**：实时显示最近20条操作记录

## 🛠️ 技术特性

- **Windows API点击**：使用SendInput API，绕过游戏保护
- **多线程设计**：点击循环在独立线程运行，界面流畅
- **全局快捷键**：使用pynput库实现系统级快捷键
- **数据持久化**：JSON格式保存配置和预设

## 📋 系统要求

- **操作系统**：Windows 7/8/10/11
- **Python版本**：3.7+ （仅源码运行需要）
- **依赖库**：
  - tkinter（Python自带）
  - pynput
  - pyautogui
  - Pillow

## ⚠️ 注意事项

1. **使用场景**：本程序仅供学习和合法用途使用
2. **游戏使用**：某些游戏可能检测并禁止自动点击工具
3. **点击频率**：建议不要设置过快的点击频率，避免系统负担
4. **快捷键冲突**：如果快捷键与其他程序冲突，请自定义修改
